# Каталог продуктов

Веб-приложение для управления каталогом товаров с разграничением доступа по ролям.

## Возможности

- Вход по логину/паролю с JWT токенами
- Три роли: администратор, продвинутый пользователь, простой пользователь
- Управление продуктами и категориями
- Поиск и фильтрация товаров
- Конвертация цены в USD по курсу НБ РБ
- Журнал действий пользователей
- Адаптивный интерфейс

## Технологии

- Backend: FastAPI, SQLAlchemy, PostgreSQL
- Frontend: HTML, CSS, JavaScript
- Контейнеризация: Docker, Docker Compose

## Быстрый старт

### 1. Клонирование репозитория

```bash
  git clone https://github.com/sold1er1K/products_catalog.git
  cd products_catalog
```
### 2. Откройте Docker Desktop

### 3. Запустите в терминале
```bash
  docker-compose up --build
```

## Доступ к приложению

| Страница         | Адрес |
|------------------|-------|
| Главная страница | http://localhost:8000 |
| Страница входа   | http://localhost:8000/login |
| Документация API | http://localhost:8000/api/docs |

## Тестовые учетные записи

| Роль | Логин | Пароль |
|------|-------|--------|
| Администратор | admin | admin123 |
| Продвинутый | advanced | advanced123 |
| Простой | user | user123 |

## Что может делать каждая роль

**Простой пользователь**
- Просмотр каталога
- Добавление и редактирование товаров
- Не видит поле "Примечание специальное"

**Продвинутый пользователь**
- Всё, что может простой
- Удаление товаров
- Управление категориями
- Видит поле "Примечание специальное"

**Администратор**
- Всё, что может продвинутый
- Управление пользователями (создание, блокировка, удаление)
- Просмотр журнала действий

## Структура API

| Метод | Endpoint | Описание |
|-------|----------|----------|
| POST | /api/auth/login | Вход |
| POST | /api/auth/logout | Выход |
| GET | /api/auth/me | Текущий пользователь |
| GET | /api/products/ | Список товаров |
| POST | /api/products/ | Добавить товар |
| PATCH | /api/products/{id} | Изменить товар |
| DELETE | /api/products/{id} | Удалить товар |
| GET | /api/categories/ | Список категорий |
| POST | /api/categories/ | Добавить категорию |
| PATCH | /api/categories/{id} | Изменить категорию |
| DELETE | /api/categories/{id} | Удалить категорию |
| GET | /api/users/ | Список пользователей (админ) |
| GET | /api/logs/ | Журнал действий (админ) |
| GET | /api/currency/usd-rate | Курс USD |
